Starting in Cyber Security? TryHackMe is your PATH

Mohamed Elleuch
July 2021
Image credit: TryHackMe

Abstract

TryHackMe is an online platform for learning cyber security and penetration testing through hands-on exercises and labs designed to teach practical skills. That has several interesting learning paths for beginners/advcanced players! For this blog we’re gonna talk about the Cyber Fundamentals learning PATH.

Publication
In TryHackMe Pre Security PATH
First Create your TryHackMe’s Account.
Enroll in the Pre-Security Path.

Path Introduction

This learning path will teach you the pre-requisite technical knowledge to get started in cyber security. To attack or defend any technology, you have to first learn how this technology works.

The Pre-Security learning path is a beginner friendly and fun way to learn the basics.

Let the learning & Fun begin!

Cyber Security Introduction

This section contains only one room Learning Cyber Security which gives a General Introduction to Cyber Security, how important some aspects are in this field and an opportunity to get your hands dirty!

Network Fundamentals

This is the heart and soul of it, It’s understanding how devices interact with each other and how data gets from point X to point Y. A strong knowledge & foundation in networking will you a very good troubleshooter whether you’re red teaming, defending or IT ops. You’ll have a general idea at first, few basics that you need to know to start building your knowledge ( What’s a protocol and few examples of it, .. I’ll not spoil more I’ll let you discover everything else by yourself).
And surely you’ll learn about the OSI Model which is one of th conceptual model that governs the computer networking ( There is the TCP/IP Model aswell). You’ll also understand what a packet or a frame and how data is divided into small pieces (PDU SDU.. ) And finally you’ll learn how to expend your network.

How The Web Works

This section is the most interesting one for me and i am sure that beginners will benefit from the amount of knowledge that they’re presenting. It covers so many aspects of the web and how it works!
This section will not only give a solid understanding but also an opportunity to practice what you have learnt!
If i started my pentesting journey with these rooms that would have saved a lot of time for me!

Linux Fundamentals

As a beginner you must learn linux or else you’re going nowhere in this field! They are many roles within cyber security that requires you have basic & fondamental understanding of linux and how it works! You don’t need to be a linux expert to excel here by all means but having a fairly good understanding about how this operating system works will greatly help in your journey!
And this section will teach almost all the basics that you need to start getting your hands dirty in the other labs in this platforme or maybe another path.
After doing this section i highly recommand trying the Linux challenges room! You’ll get the chance to challenge your skills even more and learn more tricks and tips

Windows Fundamentals

You maybe already familar with windows as most of us grew using windows! But do you really know how it works? Have you dig into the configurations and tried to play with them and understood nothing?!

You may 98% of the time use Linux as an attacker! But most of the time you can’t really know what OS you’re attacking! As we know most of the companies uses Windows and to be exact an Active Directory infrastructure. In order to attack a windows machine or trying to privesc to administrator you’ll need to understand the basics of Windows!
I don’t think you’ll need more resources about “Windows Basics” after completing this section it simply covers everything you need to know as a beginner!

What i do recommand after this section is checking the Active Directory rooms that are availble in the platform to enhance your knowledge

Thanks for reading, Hope you enjoyed my review!

Happy HACKING!
